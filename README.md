# PHP Coursework 1: Bookstore

## Description
This is a simple PHP web application built for a Week 1 Web Development assignment. It simulates a basic bookstore system where users can add new books using a form. The project demonstrates the use of PHP multidimensional arrays, POST request handling, pass-by-reference functions for genre-based discounts, and basic file logging to track entries.

## How to Run

1. **Place the files:** Copy this entire repository folder into your local server's document root:
   * For **AMPPS** or **WAMP**: Place it inside the `www` folder.
   * For **XAMPP**: Place it inside the `htdocs` folder.

2. **Start the server:** Open your local server control panel (e.g., AMPPS) and make sure the **Apache** service is running.

3. **Open the project:** Open your web browser and navigate to the following URL (adjust the folder name `PHP-COURSEWORK1-BOOKSTORE` if you saved it under a different name):
   `http://localhost/PHP-COURSEWORK1-BOOKSTORE/index.php`
